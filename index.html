Exploration of various deep neural networks for Question Answering

Status: In progress

Reading comprehension is a very important problem in natural language processing. The recently released dataset, Stanford Question Answering Dataset (SQuAD) offers a large number of questions and their answers created by crowdworkers on a set of Wikipedia articles. SQuAD compared with previous datasets, e.g. CNN/Daily Mail, provides a more challenging task, in part due to the fact that in SQuAD, unlike other datasets, answers have variable lengths and do not come from a set of candidate answers. we are working on seq2seq models with different kinds of attention mechanisms, pointer networks and etc.

Iris: Persian Named Entity Tagger

status: Finished

Recognizing and extracting exact named entities, e.g. Persons, Locations and Organizations from natural language text is called Named Entity Recognition (NER). In this paper, firstly, we are going to explain different techniques, methods, discuss design challenges, misconceptions and features which underline the development of an efficient and robust NER system. Then, we are going to introduce Iris: the Farsi named entity recognizer agent which is based on statistical distribution techniques and machine learning methods. Iris uses techniques corresponding to natural language processing and Farsi words and sentences' syntax and structure.

Alefba: Persian Optical Character Recognition System

status: In Preparation

Improvements and advancements in OCR techniques have been uneven across different languages, especially Persian. Persian written text's form, delicate word segmentation and phonetic notations in text have impeded a close to human performance; thus, making Persian OCR a very challenging path to step into. Our model takes advantage of Convolutional Layers for feature extraction, Bi-Directional LSTMs and Connectionist Temporal Classification (CTC) cost to label sequences.
Since there was no large dataset available, we created a suitable one using several different fonts. We also added noise to this artificial dataset in order to have a higher semblance to real noisy images.
The Model was first developed with Lua, and then for better integration we switched to TensorFlow for production phase. Many Companies now use this unrivaled system(as a service) in Iran.
